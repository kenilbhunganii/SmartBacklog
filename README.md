# SmartBacklog

SmartBacklog is a simple Kanban-style task management web application built using Agile methodology.

## Features

- Kanban board (To Do, In Progress, Done)
- Add tickets
- Move tickets between columns
- Delete tickets
- AI suggestion feature

## Technologies Used

- Node.js
- Express.js
- HTML
- CSS
- JavaScript
- JSON storage
- GitHub

## How to Run

1. Install dependencies:

npm install

2. Start server:

npm start

3. Open browser:

http://localhost:3000
